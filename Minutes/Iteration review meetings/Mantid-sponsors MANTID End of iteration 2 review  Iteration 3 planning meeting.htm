Aim of Iteration 2

To provide support within the framework of sufficient instrument geometry knowledge to allow algorithms to adjust for L1 and L2 distances.  This will be tested by the implementation of a convert to wavelength algorithm.

Also

        - Python interface

        - Dynamic loading of dlls and factory registration

Progress during Iteration 2

Geometry - Almost complete, just needs merging into source control

Tof2WaveLength - Complete but currently using a stub for the geometry

Python interface - working for Framework manager, base algorithm, and base workspace, but some work needs to be done to compile in windows.

Dynamic dll loading and registration - Stalled on a communication error (PMs fault).  However the two aspects were shown to work independently and we just need to bring them together.
